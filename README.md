This is the official public distribution repository and CDN for the **OmniCraft ERP** application. 

OmniCraft is a lightweight management and accounting system designed specifically for leather crafters and other individual artisans and workshops.

The core Python/Flet source code of the application is closed-source and hosted within a private repository, while compiled standalone executables, installers, and the update manifest are distributed here.

## 📥 How to Download?

### Standard Installation (Recommended)
You can download the latest stable installer (`OmniCraft_Setup.exe`) directly from the download button on the [official website](https://omnicraft-erp.com).

### Specific & Legacy Versions
If you require a specific version or need to review older builds, you can find them under the **[Releases](https://github.com/NikMurashkin/OmniCraft-ERP-Releases/releases)** tab of this repository.

---

## ⚙️ System Requirements

*   **Operating System:** Windows 10 / Windows 11 (64-bit only)
*   **Internet Connection:** Only required for checking updates, activating license keys, and performing optional workspace syncs. The database works 100% offline.
